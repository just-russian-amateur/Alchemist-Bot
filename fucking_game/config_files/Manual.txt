=================ПЕРЕЛИВАНИЕ ЖИДКОСТЕЙ=================
Здесь будет рассказано о том, как работать с исполняемым файлом и json-файлом для решения головоломки с переливанием жидкостей.

Для удобной работы с json скачайте себе один из следующих текстовых редакторов, если у вас не установлен ни один из них:
Notepad++ (самую свежую доступную версию): https://notepad-plus-plus.org/downloads/
Visual Studio Code: https://code.visualstudio.com/#alt-downloads

Колбы заполняются двигаясь слева направо и сверху вниз по скриншоту
Представьте, что ваши колбы перевернуты вниз головой (дном вверх)
и заполняете вы их начиная с днища (то есть с верхней ячейки)

Вы можете использовать любой набор цветов или любые другие назвния
Я использую: BLUE, ORANGE, YELLOW, RED, GREEN, DARKBLUE, DARKRED,
DARKGREEN, PINK, DARKPINK, LIGHTPINK, PURPLE, GRAY, LILAC
EMPTY означает пустую ячейку в колбе соответственно

Пример того, по какому принципу заполняются колбы:
       ______        ______        ______
      / GRAY \      /PURPLE\      / GRAY \
      |ORANGE|      | PINK |      | GREEN|
      |YELLOW|      |YELLOW|      |  RED |
      | BLUE |      | GREEN|      |ORANGE|      И Т.Д.

    [             [             [
      "GRAY",       "PURPLE",     "GRAY",
      "ORANGE",     "PINK",       "GREEN",
      "YELLOW",     "YELLOW",     "RED",
      "BLUE"        "GREEN"       "ORANGE"
    ]             ]             ]
      
Порядок действий:

1. Для простых уровней, где сразу известны все цвета в колбах вы можете сразу открывать json-файл для записи начального положения.

2. На более высоких уровнях потратьте несколько попыток для того, чтобы постараться открыть как можно больше цветов.
В идеальном случае, для повышения эффективности оставить 2-4 неугаданных цвета в нижних ячейках. В худшем постарайтесь оставить не-
разгаданными только нижние ячейки.

3. В json-файле вы должны редактировать количество колб в зависимости от уровня. Всегда оставляйте 2 пустых колбы. Для добавления
новой колбы вы должны вставить часть кода:
[
    "GREEN",
    "ORANGE",
    "BLUE",
    "DARK_GREEN"
]
после символов "],", перейдя на следующую строку.
Для удаления ненужной колбы вы должны удалить приведенные выше строки (все, что указано в квадратных скобках - это 1 колба).

4. Составьте список неразгаданных цветов и вставляйте их в оставшиеся ячейки наугад.

5. После того, как вы добавили состояния для всех колб сохраните изменения и запустите исполняемый файл (flasks.exe).

=================FLUID TRANSFUSION=================
This will show you how to work with the executable and json file to solve the liquid pouring puzzle.

For convenient work with json, download one of the following text editors if you do not have any of them installed:
Notepad++ (latest version available): https://notepad-plus-plus.org/downloads/
Visual Studio Code: https://code.visualstudio.com/#alt-downloads

The flasks are filled moving from left to right and from top to bottom in the screenshot
Imagine your flasks are upside down (bottom up)
and you fill them starting from the bottom (that is, from the top cell)

You can use any set of colors or any other names
I use: BLUE, ORANGE, YELLOW, RED, GREEN, DARKBLUE, DARKRED,
DARKGREEN, PINK, DARKPINK, LIGHTPINK, PURPLE, GRAY, LILAC
EMPTY means empty cell in the flask respectively

An example of the principle by which flasks are filled:
       ______        ______        ______
      / GRAY \      /PURPLE\      / GRAY \
      |ORANGE|      | PINK |      | GREEN|
      |YELLOW|      |YELLOW|      |  RED |
      | BLUE |      | GREEN|      |ORANGE|      ETC.
      
    [             [             [
      "GRAY",       "PURPLE",     "GRAY",
      "ORANGE",     "PINK",       "GREEN",
      "YELLOW",     "YELLOW",     "RED",
      "BLUE"        "GREEN"       "ORANGE"
    ]             ]             ]

The procedure:

1. For simple levels, where all the colors in the flasks are immediately known, you can immediately open the json file to record the
initial position.

2. At higher levels, spend several attempts to try to open as many flowers as possible.
Ideally, to increase efficiency, leave 2-4 unguessed colors in the lower cells. At worst, try to leave only the bottom cells unsolved.

3. In the json file, you must edit the number of flasks depending on the level. Always leave 2 empty flasks. For adding new flask you
should paste part of the code:
[
     "GREEN"
     "ORANGE",
     "BLUE"
     "DARK_GREEN"
]
after the "]" characters, going to the next line.
To remove an unnecessary flask, you must remove the lines above (everything in square brackets is 1 flask).

4. Make a list of unsolved colors and insert them into the remaining cells at random.

5. After you have added states for all flasks, save the changes and run the executable file (flasks.exe).
